Summary: Latitude Vs X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeather Api to assempble a dataset of over 500 cities.

After Assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at included: temperature, cloudiness, wind speed and humidity. This stite provides the source data and visualizations created as part of the analysis, as well as expectations and descriptions of any trends and correlations withnessed.
